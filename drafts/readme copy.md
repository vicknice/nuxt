## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm start

# Run unit tests
$ npm run test:unit

# Run end-to-end tests
$ npm run test:e2e
```
